Desarolla a traves de todas las plataformas

Aprende a desarrollar aplicaciones con Angular reutiliza tu código y potencie la creación de aplicaciones para web, web móvil, móvil nativo, escritorio nativo y cualquier otro objetivo.

VELOCIDAD Y RENDIMIENTO

Logra la máxima velocidad posible en las Plataformas Web hoy y llevalo más lejos, a través de Trabajadores web (Web Workers) y la representación del lado del servidor.

Angular te permite controlar la escalabilidad. Satisface requerimientos enormes de datos construyendo modelos de datos en RxJS, Immutable.js o algún otro modelo-de-empuje(push-model).

HERRAMIENTAS INCREÍBLES

Crea funciones rápidamente con plantillas simples y declarativas. Extienda el lenguaje de plantilla con sus propios componentes y use una amplia gama de componentes existentes. Obtenga ayuda inmediata y comentarios específicos de Angular con casi todos los IDE y editores. Todo esto se une para que pueda concentrarse en crear aplicaciones increíbles en lugar de intentar que el código funcione.

AMADO POR MILLONES

Desde el prototipo hasta la implementación global, Angular ofrece la productividad y la infraestructura escalable que admite las aplicaciones más grandes de Google.

PRUEBALO AHORA

Explora las capacidades de Angular con una aplicación de muestra preparada. No se requiere configuración.
